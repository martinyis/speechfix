  <action>
    1. Create the `server/` directory at project root.

    2. Initialize `server/package.json` with:
       - name: "reframe-server"
       - version: "0.1.0"
       - private: true
       - "type": "module"
       - scripts:
         - "dev": "tsx watch src/index.ts"
         - "start": "tsx src/index.ts"
         - "db:push": "drizzle-kit push"
         - "db:generate": "drizzle-kit generate"
         - "db:migrate": "drizzle-kit migrate"

    3. Install dependencies via npm (NOT yarn):
       - Production: fastify@^5, @fastify/multipart@^9, @fastify/cors, drizzle-orm, postgres, openai@^6, @anthropic-ai/sdk, zod, dotenv
       - Dev: typescript@^5.7, @types/node, drizzle-kit, tsx

    4. Create `server/tsconfig.json` with target ES2022, module ES2022, moduleResolution bundler, strict true, esModuleInterop true, skipLibCheck true, outDir dist, rootDir src, resolveJsonModule true, declaration true. Include src/**/*. Exclude node_modules and dist.

    5. Create `server/.env` with:
       DATABASE_URL=postgresql://localhost:5432/reframe
       OPENAI_API_KEY=placeholder
       ANTHROPIC_API_KEY=placeholder
       PORT=3000

    6. Create `server/.env.example` with the same keys but empty values for OPENAI_API_KEY and ANTHROPIC_API_KEY. Keep DATABASE_URL=postgresql://localhost:5432/reframe and PORT=3000 as defaults.

    7. Create `server/.gitignore` containing: node_modules/, dist/, .env, *.m4a, *.wav

    8. Create `server/src/index.ts` -- Fastify server entry point:
       - Import 'dotenv/config' at the very top
       - Import Fastify from 'fastify'
       - Import fastifyCors from '@fastify/cors'
       - Import fastifyMultipart from '@fastify/multipart'
       - Import { db } from './db/index.js'
       - Import { sql } from 'drizzle-orm'
       - Create Fastify instance: const app = Fastify({ logger: true })
       - Register fastifyCors with { origin: true }
       - Register fastifyMultipart with { limits: { fileSize: 25 * 1024 * 1024 } }
       - Add GET /health route returning { status: 'ok', timestamp: new Date().toISOString() }
       - In the start function: listen on { port: Number(process.env.PORT) || 3000, host: '0.0.0.0' }
       - After listen succeeds, run: await db.execute(sql`SELECT 1`) and log 'Database connected'
       - Wrap in try/catch, log error and process.exit(1) on failure

    9. Create `server/src/db/schema.ts` with three tables using drizzle-orm/pg-core:

       sessions table:
       - id: serial('id').primaryKey()
       - transcription: text('transcription').notNull()
       - durationSeconds: integer('duration_seconds').notNull()
       - analysis: jsonb('analysis') -- nullable, Phase 1 stores only transcription
       - createdAt: timestamp('created_at').defaultNow().notNull()

       corrections table:
       - id: serial('id').primaryKey()
       - sessionId: integer('session_id').references(() => sessions.id, { onDelete: 'cascade' }).notNull()
       - originalText: text('original_text').notNull()
       - correctedText: text('corrected_text').notNull()
       - explanation: text('explanation')
       - correctionType: text('correction_type').notNull()
       - createdAt: timestamp('created_at').defaultNow().notNull()

       fillerWords table:
       - id: serial('id').primaryKey()
       - sessionId: integer('session_id').references(() => sessions.id, { onDelete: 'cascade' }).notNull()
       - word: text('word').notNull()
       - count: integer('count').notNull()
       - createdAt: timestamp('created_at').defaultNow().notNull()

       Export all three tables.

    10. Create `server/src/db/index.ts`:
        - Import 'dotenv/config'
        - Import { drizzle } from 'drizzle-orm/postgres-js'
        - Import postgres from 'postgres'
        - Import * as schema from './schema.js' (use .js extension for ESM)
        - const connectionString = process.env.DATABASE_URL!
        - const client = postgres(connectionString)
        - export const db = drizzle(client, { schema })

    11. Create `server/drizzle.config.ts`:
        - Import 'dotenv/config'
        - Import { defineConfig } from 'drizzle-kit'
        - Export default defineConfig with schema: './src/db/schema.ts', out: './drizzle', dialect: 'postgresql', dbCredentials: { url: process.env.DATABASE_URL! }

    12. Create the PostgreSQL database: run `createdb reframe` (ignore error if it already exists).

    13. Push schema to database: run `cd server && npx drizzle-kit push` to create all three tables.
